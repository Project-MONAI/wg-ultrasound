# wg-ultrasound / Data and Tutorials

This repository holds the work-in-progress material for the ultrasound
working group's Data and Tutorials subgroup.

## Datasets

We manage a [website](https://ultrasound-open-access.nidusai.ca/) for searching through all our datasets, controlled from a google spreadsheet. Please email amcarth1@ualberta.ca for permissions.

Please email amcarth1@ualberta.ca or leave an issue on GitHub for new datasets! https://github.com/nidus-lab/open-source-imaging-data-sets/issues
